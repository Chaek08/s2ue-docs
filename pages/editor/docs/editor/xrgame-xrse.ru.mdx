# Перезагрузка XrGame и XrSE_Factory

Перезагрузка XrGame и XrSE_Factory делается одной кнопкой

![...](./xrgame-xrse.ru/img/xr-reload.png)

***

Если есть потребность перезагрузить, обновить список объектов или обновить конфиги, то эта кнопка для вас.
Она полностью перезагружает `system.ltx`, `game.ltx`

Также перезагружает `se_factory` включая его скрипты.
У XrGame скрипты перезагружаются каждый запуск PIE (Play In Editor)!
