# LaTeX

Ми використовуємо [KaTeX](https://katex.org/) для відображення виразів LaTeX безпосередньо у MDX.

## Приклад

Наприклад, наступний код Markdown:

```md filename="page.mdx"
**Теорема Піфагора**: $a=\sqrt{b^2 + c^2}$.
```

Буде відображено таким чином:

<div className="mt-6 rounded border border-gray-200 p-4 dark:border-gray-900">
  **Теорема Піфагора**: $a=\sqrt{b^2 + c^2}$.
</div>

Ви все ще можете використовувати [синтаксис Markdown і MDX](markdown) в тому ж рядку, де знаходиться ваш вираз LaTeX.

<Callout>
  Якщо ви хочете відобразити символ `$` у вашому контенті, а не інтерпретувати його як вираз, ви можете екранувати його зворотним слешем (`\`). Наприклад, `\$e = mc^2\$` буде відображено як \$e = mc^2\$.
</Callout>
