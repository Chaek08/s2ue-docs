# Как добавить или изменить информацию

Для редактирования документации создайте Issue или Pull request на странице [GitHub](https://github.com/psiget/s2ue-docs)

## Создание Pull request

<Steps>

### Выберите необходимый файл для редактирования

### Нажмите на иконку редактирования

![...](./doc-public.ru/img/icon-edit.png)

### Создайте Форк

![...](./doc-public.ru/img/create-fork.png)

### Внесите необходимые правки и нажмите `Commit changes`

![...](./doc-public.ru/img/commit-changes.png)

### Нажмите `Propose changes`

![...](./doc-public.ru/img/propose-changes.png)

### Нажмите `Create pull request`

![...](./doc-public.ru/img/create-pull-request.png)

### Нажмите снова `Create pull request`

![...](./doc-public.ru/img/create-pull-request-x2.png)

### Готово! Осталось только подождать!

</Steps>

Спасибо за ваш вклад! Как только наши модераторы освободятся, так сразу займутся обработкой ваших правок!
