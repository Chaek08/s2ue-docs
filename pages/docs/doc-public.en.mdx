# How to Add or Modify Information

To edit the documentation, create an Issue or Pull Request on the [GitHub](https://github.com/psiget/s2ue-docs) page.

## Creating a Pull Request

<Steps>

### Select the file you want to edit

### Click on the edit icon

![...](./doc-public.ru/img/icon-edit.png)

### Create a Fork

![...](./doc-public.ru/img/create-fork.png)

### Make the necessary changes and click `Commit changes`

![...](./doc-public.ru/img/commit-changes.png)

### Click `Propose changes`

![...](./doc-public.ru/img/propose-changes.png)

### Click `Create pull request`

![...](./doc-public.ru/img/create-pull-request.png)

### Click `Create pull request` again

![...](./doc-public.ru/img/create-pull-request-x2.png)

### Done! Now, just wait!

</Steps>

Thank you for your contribution! As soon as our moderators are available, they will start processing your changes!
