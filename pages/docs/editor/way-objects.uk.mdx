# Як працювати з об'єктами шляху (Way Objects)

## Об'єкти шляху (Way Objects)
Спочатку потрібно створити об'єкт шляху. Знайдіть у секції "Place Actors" об'єкт "Stalker Way Object".

![Об'єкт шляху](./way-objects.ru/img/way-object.png)

Потім перейдіть у режим редагування "Way Points Editor" (Редактор точок шляху).

![Редактор точок шляху](./way-objects.ru/img/way-points-editor.png)

### Опис режиму редагування
Цей режим повністю аналогічний оригінальному редактору.

![Опис режиму редагування](./way-objects.ru/img/description-edition-mode.png)

## Додаткова інформація

Об'єкти шляху є частиною спавну, тому під час їх редагування спавн стає застарілим і потребує перезбірки.
Процес перезбірки може бути автоматичним або вручну, залежно від налаштувань.
Для отримання інформації про роботу з об'єктами спавну, прочитайте [Як працювати з об'єктами спавну (Spawn Objects)](./spawn-objects)

Ви можете включати або вимикати відображення точок шляху в меню "show" (показати).

![Додаткова інформація](./way-objects.ru/img/additional-information.png)
